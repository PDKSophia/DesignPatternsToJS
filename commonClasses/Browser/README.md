# Browser Classes

## 主要功能

- 获取主流浏览器名称
- 包含主流浏览器
  - Chrome 浏览器 ✅
  - FireFox 浏览器 ✅
  - Safari 浏览器 ✅
  - Edge 浏览器 ✅
  - IE 浏览器（5~11）✅
  - QQ 浏览器 ✅
  - UC 浏览器 ✅
  - Opera 浏览器 ✅
  - Sougou 浏览器 ✅
  - Baidu 浏览器 ✅
  - 遨游浏览器 ✅
  - 猎豹浏览器 ✅
  - 夸克浏览器 ✅
  - 世界之窗浏览器 ✅
  - 360 浏览器 ✅
